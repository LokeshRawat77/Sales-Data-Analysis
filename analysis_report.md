# Sales Data Analysis Report

## Project Overview
This project analyzes a sales dataset using Python and Pandas. The goal is to calculate total sales, identify the best-selling product, and generate a clean report.

## Dataset Columns
- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

## Steps Performed
1. Loaded CSV file using Pandas
2. Explored rows, columns, and missing values
3. Handled missing values
4. Removed duplicate records
5. Calculated sales metrics
6. Created a formatted report

## Metrics Calculated
- Total Revenue
- Total Quantity Sold
- Average Sale Value
- Best-Selling Product
- Highest Revenue Product
- Top Performing Region

## Result
The program successfully analyzes sales data and prints a clean report in the terminal.

## How to Run
```bash
pip install -r requirements.txt
python sales_analysis.py
